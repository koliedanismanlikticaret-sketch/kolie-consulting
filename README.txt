Kolie Consulting - site refactor (HTML/CSS/JS séparés)

1) Ouvrir index.html dans un navigateur.
2) Si vous avez une vidéo, mettez-la dans assets/media/video-presentation.mp4
   puis dans index.html remplacez: src="video-presentation.mp4" par src="assets/media/video-presentation.mp4"
3) Déploiement (GitHub Pages / Cloudflare Pages / Vercel): uploadez tout le dossier.

Structure:
- index.html
- assets/css/styles.css
- assets/js/translations.js
- assets/js/main.js
- assets/media/ (vos fichiers locaux: vidéo, images, logo...)
